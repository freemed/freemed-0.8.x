FreeMED Error 

This is the index.html, but you need to view index.php as the default.

If this is all you see (if there is no login screen far above this), then you have successfully gotten Apache to work, but not PHP. FreeMED requires that you have PHP and the PHP MySQL extention to run. More information is available at apache.org or php.net.

If you have PHP installed and working already, you need to make sure that "index.php" is the first filename listed in the DirectoryIndex directive in your Apache configuration.

For more information, please visit either freemedsoftware.org or freemedsoftware.com.
