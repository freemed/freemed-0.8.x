This is the Administration Menu of FreeMED. It can be reached either through the Main Menu or via the menu bar on the left side of the screen using the default theme.

  • Import/Export Databases - This gives you the ability to import and export FreeMED's low-level databases to CSV type files, which can be read by most programs. This is good for exchanging data with other programs that cannot use FreeMED's XML-RPC capabilities.
  • Module Information - Provides a list of the installed loadable modules on the system and basic information about them.
  • Reinitialize Database - This is the first step that you should take when creating a new installation of FreeMED. It wipes all of the tables in the database and has the option to reload the stock data tables that come with FreeMED. This should never be used on a running system, or else complete loss of data will occur.
  • %%Update Config,admin,cfgform%% - Set the parameters that FreeMED uses to define various portions of its behavior.
  • User Maintenance - Add and remove users from the system. Please note that you cannot remove the root user from the system.
Some of the functions in this menu require superuser access. The information for your "superuser" account (by default called "root") should never be given out, and should be guarded closely, as it has an incredible amount of power to modify the system.
